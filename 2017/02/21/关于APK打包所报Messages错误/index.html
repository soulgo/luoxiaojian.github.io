Contents
  1. 1. 打包APK提示错误
  • 添加multiDexEnabled true解决，原因是引用了多个libraries文件
    1. 1. log框报以下错误，用同样的方法解决

    • 本篇总结了APK打包所报Messages错误的问题..

    打包APK提示错误

    Errors while building APK. You can find the errors in the 'Messages' view.

    添加multiDexEnabled true解决，原因是引用了多个libraries文件

    defaultConfig {

    .....
    multiDexEnabled true
}
