SharedPreferences

用法：
SharedPreferences对象本身只能获取数据而不支持存储和修改，存储修改是通过Editor对象实现的。

存放：

1.获得SharedPreferences 的实例对象，通过getSharedPreferences()传递存储时的名称和模式

2.获得Editor 的实例对象，通过SharedPreferences 的实例对象的edit()

3.存入数据用Editor的putXXX()  [存放什么数据就put那个数据的类型,支持Long、Int、String、Boolean]

4.提交修改的数据用Editor的commit()

读取：

1.获得SharedPreferences 的实例对象，通过getSharedPreferences()传递存储时的名称和模式

2.读取数据，通过SharedPreferences 的实例对象的getXXX()       [读取什么类型的数据就get那个类型]

定义SharePreference的四个步骤：

1、通过getSharePreference(key，mode) 获得对象
2、利用ShareLPreference的editor()方法来获取editor对象
3、利用editor的putString()方法将键值对数据存储进去
4、利用editor的commit()方法将数据提交到SharePreference内

调用SharePreference的几个步骤：

1、通过getSharePreference方法获得SharePreference对象
2、使用对象的getString(key, default value)方式来获得SharePreference中的数据
这里需要注意的是，当SharePreference中没有这个key对应的value的时候，它也不会报错，会返回一个none，所以，建议事先定义一个default value值，方便调试。

在点击登陆按钮出添加一个调用保存密码的方法：

saveUserInfo(username, password);
此处的username和password就是你所在的用户名和密码

 username = log_user.getText().toString().trim();
 password = log_pass.getText().toString().trim();

保存用户的登陆记录

private void saveUserInfo(String username,String password){
    SharedPreferences sp = getSharedPreferences("UserInfo",0);
    SharedPreferences.Editor editor = sp.edit();
    editor.putString("username",username);
    editor.putString("password",password);
    editor.commit();
}
