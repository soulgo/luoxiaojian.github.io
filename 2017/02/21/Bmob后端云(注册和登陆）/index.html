关于Bmob后端云的jar文件的错误

今天试着用了一下Bmob的后端云，很强大，但官方文档很坑，自己摸索了好久，请教了一些人，终于弄好了。

这是主框架，主要注意红框里的类容，so文件必须要配置，为了简便起见，so文件和jar放在一起，Android Studio自动识别，

第二个红框里的是app的buid.gradle文件中添加SO库目录配置：

android {
 sourceSets {
     main.jniLibs.srcDirs = ['libs']
        }
 }
