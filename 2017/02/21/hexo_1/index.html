Hexo（一）博客搭建图文教程：本地搭建

本篇总结了如何运用Hexo搭建博客（一）..

工具准备

你需要准备好以下软件：
    - Node.js环境
    - Git
    - MarkDownPad2

安装工具（Window)

配置Node.js环境

下载Node.js安装文件：

     Windows 32-bit
    -  https://nodejs.org/dist/v4.2.3/node-v4.2.3-x86.msi

     Windows 64位-bit
    -  https://nodejs.org/dist/v4.2.3/node-v4.2.3-x64.msi

根据自己的Windows版本选择相应的安装文件，安装都是傻瓜式的安装，next按下去就行了！

验证Node安装是否成功

打开命令行窗口（在运行窗口输入cmd），输入
node -v
npm -v
开是否出现以下界面，如果出现以下界面，说明Node.js配置成功

配置Git环境

下载Git安装文件
    Git-2.6.3-64-bit.exe
        http://dwz.cn/3Oa3KK
安装步骤也是next一路下去。

验证Git安装是否成功

同样打开cmd命令行窗口，输入：
    git --version
看是否出现以下界面。出现即代表成功！

注册Github账号

如果本来有账号的可以跳过这个，
在浏览器中输入https://github.com/
如果你没有科学上网的话进不去这个网站，因为某些原因，你需要学习一下如何科学上网，学计算机的这是必备技能，网上有很多方法，
这里不在叙述。按照步骤注册好自己的用户名和密码。（切记注册完之后在自己的注册邮箱里进行确认注册，不然的话可能影响自己后面的步骤。

创建代码库

登陆之后，点击页面右上角的加号，选择New repository：如图所示

进入代码库创建页面：
在Repository name下填写yourname.github.io，Description (optional)下填写一些简单的描述（不写也没有关系），如图所示：
注意：这里的yourname是你的github用户名，不要写错了！

正确创建之后，你将会看到如下界面：

配置Hexo

安装Hexo

在自己认为合适的地方创建一个文件夹，然后在文件夹空白处鼠标右键找到Git Bash Here，然后点击在此处打开命令行窗口。在命令行中输入：npm install hexo-cli -g
你将会看到

然后输入
npm install hexo --save
输入以下命令查看是否安装好了
hexo -v

最后初始化和安装npm(安装可能需要三十秒）
hexo init
npm install

首次体验Hexo

同样的操作输入命令：
hexo g
hexo s
在浏览器中打开http://localhost:4000/，你将会看到：
到此为止，Hexo在本地的配置已经全都结束了。
